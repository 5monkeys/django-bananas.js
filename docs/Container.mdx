# Container

The `Container` component centers its `children` within a fixed width that can be defined using `overrides` in your theme. Container is used in built-in components like `TitleBar`, `ToolBar` and `Content` and is therefore rarely needed stand-alone.

## Usage

```jsx
import themes from "django-bananas/themes";

const myTheme = themes.default.extend({
  overrides: {
    BananasContainer: {
      root: {
        width: "100%",
        maxWidth: 500,
        margin: "0 auto",
      },
    },
  },
});
```
